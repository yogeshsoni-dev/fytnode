# Recommended Libraries

## Core
- `expo` / `expo-router` or `@react-navigation/native` + stacks/tabs
- `typescript`
- `axios`
- `@tanstack/react-query`
- `zustand`
- `react-hook-form`
- `zod`

## Storage & Security
- `expo-secure-store` (tokens)
- `@react-native-async-storage/async-storage` (non-sensitive preferences)
- optional `react-native-mmkv` for high-performance key-value state

## UX / Media
- `expo-image-picker`
- `expo-file-system`
- `react-native-reanimated`
- `react-native-gesture-handler`

## Quality / Ops
- `@sentry/react-native`
- `jest`, `@testing-library/react-native`, `detox`
- `eas-cli` for build/update pipelines
